## 2. 优化策略

### 2.1 信息架构重构

**核心理念**: 从"功能列表" → "工作流导向"

用户不是来"用某个功能"的，是来"完成一个任务"的。

```
旧架构:          新架构:
生成              开始创作 (统一入口)
项目              我的工作台
库                资源中心
模板              模板市场
优化              工具箱
导出              工具箱
设置              设置
关于              关于
```

**导航简化** (侧边栏):
| 优先级 | 条目 | 图标 | 说明 |
|--------|------|------|------|
| P0 | **开始** | Sparkles | 主入口：输入需求 → 生成提示词 |
| P0 | **工作台** | LayoutGrid | 项目 + Clarify + 优化历史 |
| P1 | **资源** | Library | 提示词库 + 模板 |
| P1 | **工具箱** | Wrench | Optimizer + Export + 其他工具 |
| P2 | 设置 | Settings | — |
| P2 | 关于 | Info | — |

### 2.2 首屏重设计 (Home)

**问题**: 当前 Home 就是一个输入框，用户不知道能做什么。

**优化方案**:

```
┌─────────────────────────────────────────┐
│  TipAi — AI 提示词工作流平台            │  ← 品牌 + 定位
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 🔍 描述你的需求...              │   │  ← 主输入框（更大更突出）
│  │                                 │   │
│  │   /视频脚本  /小红书文案  /代码  │   │  ← 快捷指令标签
│  └─────────────────────────────────┘   │
│                                         │
│  ┌──────┐ ┌──────┐ ┌──────┐           │
│  │ 🎬   │ │ 📝   │ │ 💻   │           │  ← 场景卡片
│  │视频  │ │内容  │ │开发  │           │
│  └──────┘ └──────┘ └──────┘           │
│                                         │
│  最近项目 → [项目1] [项目2] [项目3]     │  ← 快捷入口
│                                         │
└─────────────────────────────────────────┘
```

**关键改进**:
1. **Hero 区域** — 明确产品定位（不是"生成"，是"AI 提示词工作流"）
2. **输入框增强** — 支持 `/` 快捷指令，更大更易用
3. **场景卡片** — 降低首次使用门槛
4. **最近项目** — 快速续作

## 4. 组件规范

### 4.1 按钮层级

| 层级 | 样式 | 用途 |
|------|------|------|
| **Primary** | bg-blue + white text + shadow | 主操作（创建、提交） |
| **Secondary** | bg-gray + dark text | 次操作（取消、返回） |
| **Ghost** | transparent + border | 辅助操作（设置、筛选） |
| **Destructive** | bg-red + white text | 危险操作（删除） |

### 4.2 卡片层级

| 层级 | 圆角 | 阴影 | 用途 |
|------|------|------|------|
| **Card** | md (0.625rem) | shadow-sm | 内容容器 |
| **Card Hover** | md | shadow-md + scale(1.01) | 可交互卡片 |
| **Card Active** | md | shadow-lg + ring | 选中状态 |

### 4.3 输入框规范

```
默认: bg-white border-gray-200 rounded-xl
Focus: ring-2 ring-blue-500 border-transparent
Error: border-red-300 bg-red-50
Success: border-green-300 bg-green-50
Disabled: bg-gray-100 text-gray-400
```

## 5. 交互优化

### 5.1 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 全局搜索 |
| `Cmd/Ctrl + N` | 创建新项目 |
| `Cmd/Ctrl + O` | 打开优化器 |
| `Cmd/Ctrl + E` | 打开导出 |
| `Cmd/Ctrl + /` | 显示快捷键帮助 |
| `Esc` | 关闭弹窗/返回 |

### 5.2 加载状态

```
骨架屏 > 旋转图标 > 渐进式内容加载

原则:
1. 等待 < 300ms: 直接显示
2. 等待 300ms-1s: 骨架屏
3. 等待 > 1s: 骨架屏 + 进度提示
```

### 5.3 错误处理

```
网络错误:    "连接不稳定，已自动重试 (1/3)"
API 错误:    "服务暂时不可用，请稍后重试"
输入错误:    即时验证，红字提示
未知错误:    "出了点问题，错误代码: XXX"
```
