## 通用规范

### 设计系统遵循
- 使用现有 CSS 变量: `--primary`, `--background`, `--card` 等
- Tailwind 扩展: `apple-shadow`, `apple-font`, `ease-apple`
- 圆角: 默认 `rounded-xl` (0.625rem)
- 间距: 优先使用 `space-y-4`, `p-6`, `gap-4`

### 动画规范
```css
--ease-apple: cubic-bezier(0.25, 0.1, 0.25, 1);
--duration-fast: 150ms;
--duration-normal: 250ms;
```
- 卡片 hover: `transition-all duration-250 ease-apple hover:shadow-md hover:scale-[1.01]`
- 按钮 active: `active:scale-[0.98]`
- 页面切换: `fade-in` 或 `slide-up`

### 响应式要求
- `sm` (640px): 手机横屏适配
- `md` (768px): 平板适配
- `lg` (1024px): 桌面标准
- 移动端侧边栏: 可折叠或底部导航

### 无障碍 (WCAG)
- 所有按钮有 `aria-label`
- 输入框有关联 `label`
- 颜色对比度 ≥ 4.5:1
- 支持键盘导航
